## 프로젝트 개요
- 사용자 맞춤형 건강 상담 챗봇
- 대화형 UI/UX를 통한 자연스러운 상담 경험 제공
- 실시간 건강 정보 및 영양제 추천

## 기술 스택
- React 18.x
- TypeScript 4.x
- Styled-components
- Recoil (상태 관리)
- Vite (빌드 도구)
- GitHub Actions (CI/CD)
